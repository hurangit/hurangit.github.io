步骤 1：调整文章图片布局

首先你需要确保文章的图片位置为左侧布局。
打开主题配置文件，搜索封面顯示的位置。

将 position 改为 left。
同时设置 hexo 的页面展示文章数首页文章数最好大于 20 篇。
目前已修复，不过最好多一点不然侧边栏会显得过长。

步骤 2：接入 CSS

打开主题配置文件
搜索到”inject:”设置处
原生添加以下代码或下载 css 从本地导入：

链接进入后，你会发现，布局神奇地变成了双栏布局。

步骤 3：修改 pug

如果你想让发布时间、分类、标签不在一行显示可以修改 pug。
